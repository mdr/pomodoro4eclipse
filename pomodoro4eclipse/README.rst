Pomodoro Timer
==============

A timer plugin for Eclipse intended for use with the `Pomodoro Technique`_.

.. _Pomodoro Technique: http://www.pomodorotechnique.com

Window -> Show View -> Other

Pomodoro -> Pomodoro
